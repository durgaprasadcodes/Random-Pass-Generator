# 🔐 Random Password Generator

A **secure, modern, and responsive Random Password Generator** built using **HTML, CSS, and JavaScript**.  
This project demonstrates **JavaScript logic, conditional handling, and DOM manipulation**.

----

## 🌐 Live Demo
👉 **[Click here to view live demo](https://durgaprasadcodes.github.io/Random-Pass-Generator/)**

----

## ✨ Features

- 🔢 Generate strong random passwords  
- 🔠 Include uppercase letters  
- 🔡 Include lowercase letters  
- 🔣 Include numbers  
- 🔐 Include special characters  
- 📋 One-click copy to clipboard  
- 📱 Fully responsive UI  

----

## 🛠️ Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling & responsiveness  
- **JavaScript (ES6)** – Password logic & events
